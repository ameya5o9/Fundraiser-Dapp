# Features

• Fundraiser Dapp solves the problem of transparency and accountability in fundraising. <br>
• Decentralized nature of Fundraiser Dapp ensures that there is no central authority or single point of failure that can be corrupted or manipulated <br>
• Using smart contracts to automate the process of creating and managing campaigns  <br>
• Donors can track their contributions in real-time. <br>
